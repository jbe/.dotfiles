**To install:**

1. Fork this repo on Github
2. Paste this into a terminal:

    cd ~
    echo "Enter your name on Github:"; git clone git@github.com:`read REPLY; echo $REPLY`/.dotfiles.git;

3. Paste this into a terminal:

    sh ~/.dotfiles/install
